## How to use

### Windows

  ```bash
  mkdir -p c:/work
  cd c:/work
  git clone https://github.com/hei-synd-did/did-docs.git
  cd did-docs
  ```

### Linux

  ```bash
  # clone repo
  mkdir -p ~/work
  cd ~/work
  git clone https://github.com/hei-synd-did/did-docs.git
  cd did-docs
  ```
